# Roadmap

## Near-term
- Full SSE/WebSocket streaming for trace and token updates.
- Indexing pipeline with pgvector embeddings.
- Prompt Studio version history UI.
- Rich Monaco-based diff and template editor.

## Mid-term
- Additional inference adapters: **SGLang**, **Ollama**.
- Remote provider plugins (OpenAI/Anthropic) with strict opt-in.
- Advanced scheduling and recurring jobs.

## Long-term
- Multi-agent collaboration modes.
- Human-in-the-loop approvals per sensitive action.
- Cost/latency optimization across model roles.
